Applying different style sheets

The Additional CSS Styles directory includes 4 css files. To change the default syntax highlighter:

  • 1- Remove the Statax.css file
  • 2- Copy any of the Sunset.css, Daring.css, or Wrangler.css files to the directory where the JavaScript files are
  • 3- Rename the new CSS file to Statax.CSS
